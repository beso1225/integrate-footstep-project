# 足跡表示システム

## 概要

- 遠くにいる身近な人の存在感を感じるための足跡表示システムの作成．

## 仕様

### 屋内

- カメラを用いて以下を検出
    - 人
    - その人の足の動き -> 位置情報を出力
    - 表情や動き方 -> 感情情報の出力

### 屋外

- アプリを用いて以下を検出
    - 加速度変化による感情検出 -> 感情情報の出力
    - 位置情報...?

### 表示

- 複数台のプロジェクターを用いて足跡を地面に投影
    - 足跡の色やエフェクトを感情情報に応じて変化させる
    - プロジェクタで表示される映像に補正を掛けることで、足跡の位置を正確に地面に投影する
    - 足跡の位置情報をもとに、足跡の動きを（概ね）リアルタイムで表示する

## 現状

### 屋内
- カメラを用いた行動検出は完了
- 感情検出も実装済
- 2つの統合を実行中

### 屋外
- 位置情報は厳しそう -> 歩幅を使うかな？
- 加速度変化による感情検出は実装済

### 表示
- プロジェクタの位置補正は完了
- 足跡の表示機能の全体的な統合が未完了
